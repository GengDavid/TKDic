		<div class="text">
			<div class="pic"><img src="splash.png" width="100%" height="100%"><p>应用启动页，随机产生一张国家卡片</p></div>
			<p>TKDic三国辞典分为共三个功能模块：主界面、收藏夹和自定义人物，点击应用底部的导航栏，即可进入不同的功能界面。</p>
			<p></p>

			<h2>一、主界面</h2>
			<br>
			<div class="pic" style=" width: 850px; margin: 0 auto;">&nbsp;&nbsp;<img src="main.png" width="32%" height="100%">&nbsp;<img src="mainctrlist.png" width="32%" height="100%">&nbsp;<img src="mainctrprofile.png" width="32%" height="100%"><p>应用主界面</p></div>
			<p>在主界面中，点击国家卡片可以查看相应的人物列表，点击列表项可以查看人物信息。在人物信息卡片中，向右滑动卡片即可以回到人物列表。若要显示国家卡片，点击导航栏中的“主界面”<sup>*按钮1</sup>&nbsp;即可回到国家卡片界面。</p>

			<p></p>
			<h2>二、收藏夹</h2>
			<br>
			<div class="pic" style=" width: 850px; margin: 0 auto;">&nbsp;&nbsp;<img src="marklist.png" width="32%" height="100%">&nbsp;<img src="markprofile.png" width="32%" height="100%">&nbsp;<img src="markprofile2.png" width="32%" height="100%"><p>收藏夹界面</p></div>
			<p>点击底部导航栏中的“收藏夹”<sup>*按钮2</sup>&nbsp;即可进入收藏夹界面。收藏夹界面中将显示所有已收藏的人物列表，点击列表项即可查看人物详细信息。在人物卡片中，点击左下角的“收藏”按钮即可收藏人物<sup>*收藏后该按钮将变为彩色</sup>&nbsp;，再次点击可以取消收藏<sup>*取消收藏后该按钮将变为灰色</sup>。</p>

			<p></p>
			<h2>三、自定义人物</h2>
			<br>
			<div class="pic" style=" width: 850px; margin: 0 auto;">&nbsp;&nbsp;<img src="privatelist.png" width="32%" height="100%">&nbsp;<img src="privatecard1.png" width="32%" height="100%">&nbsp;<img src="privatecard2.png" width="32%" height="100%"><p>自定义人物界面</p></div>
			<p>点击底部导航栏中的“自定义人物”<sup>*按钮3</sup>&nbsp;即可进入自定义人物界面。收藏夹界面中将显示所有自定义添加的人物列表，点击列表项即可查看人物详细信息。自定义添加的人物可以被修改。点击人物详情卡片中右上角的“编辑”按钮，即可修改人物信息<sup>*对于不可编辑的人物该按钮为灰色</sup>&nbsp;。</p>

			<p>点击自定义人物列表<sup>*图1</sup>&nbsp;右上角的添加按钮<sup>*'+'号</sup>&nbsp;，即可在人物编辑卡片中添加自定义人物。</p>

			<div class="pic" style=" width: 850px; margin: 0 auto;">&nbsp;&nbsp;<img src="editcard.png" width="32%" height="100%">&nbsp;<img src="editcard2.png" width="32%" height="100%">&nbsp;<img src="editcard3.png" width="32%" height="100%"><p>人物编辑卡片</p></div>
			<p>在人物编辑卡片中，点击文本框即可输入人物信息。点击右上角的确认按钮可以检查输入的合法性，如果输入有误，点击文本框右侧的感叹号即可查看错误信息。点击人物编辑卡片中的头像，可以从本地相册中选择人物头像并添加到人物信息中，此处允许添加任意本地图片。</p>
			<p>最后，如果输入合法性检查通过，点击右上角的确认按钮即可保存人物信息。点击编辑卡片的左下角的退出按钮可以取消编辑。</p>

			<p></p>
			<h2>四、搜索框</h2>
			<br>
			<div class="pic" style=" width: 850px; margin: 0 auto;">&nbsp;&nbsp;<img src="mainsearch.png" width="32%" height="100%">&nbsp;<img src="marksearch.png" width="32%" height="100%">&nbsp;<img src="privatesearch.png" width="32%" height="100%"><p>在不同的模块中搜索“刘”的结果</p></div>
			<p>在任意界面中，点击顶部的搜索按钮或者点击搜索框即可进行人物搜索。本应用支持模糊搜索，输入单个字即可搜索出所有姓名中含有该字符的人物，在不同的模块中，只会搜索出相应模块中的人物。如上图，在主界面中可以在所有人物中搜索，在收藏夹中仅在已经收藏的人物中搜索，在自定义人物中仅在所有自定义人物中搜索。</p>
